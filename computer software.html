What is Computer Software?

Software is a set of programs that enables the hardware to perform a specific task. It is abbreviated as 'SW'. All the programs that run the computer are software. A program is a sequence of instructions written to solve a particular problem. Software is an intangible resource. You can’t hold it in your hands, While Hardware consists of tangible resources such as mice, keyboards, USB ports, CPUs, memory, printers, and so on.

There are two major types of software:

  • System software
  • Application software

1. Sytem Software

System software is a collection of programs designed to helps the user, hardware, and application software to interact and function together. System software creates an environment or platform for other software and applications to work in. It is the main software that runs the computer. When you turn on the computer, it activates the hardware and controls and coordinates their functioning. It serves as the interface between the end user and the hardware.

System software is generally prepared by the computer manufacturers. These software products comprise of programs written in low-level languages, which interact with the hardware at a very basic level. Types of system software include;

i) Operating System


Operating system is a collection of software that manages resources and provide common services for other software that runs "on top" of it. It is the system software that works as an interface to enable the user to communicate with the computer. Every device, whether a desktop, laptop or mobile phone requires an operating system to provide the basic functionality to it. There are various types of operating system such as real-time, embedded, distributed, multiuser, single-user, internet, mobile, and many more. It is important to consider the hardware specifications before choosing an operating system

For desktops computers, laptops and tablets, the types of operating systems used are;

  • Microsoft windows
  • Mac (For apple devices only)
  • Linux

For smartphones, the types of operating systems used are;

  • Google's Android
  • Apple's iOS
  • Windows phone OS

ii) Utility


Utility software manages the performance of your hardware and application software installed on your computer, to ensure they work optimally. It provides support to the computer infrastructure. Examples of utility software include:

  • Disk cleaner (eg CCleaner)
  • Data backup software(eg Norton ghost )
  • File compressor (eg WinRAR )
  • Antivirus and security software (eg Norton antivirus )
  • Disk defragmentation software (eg Defraggler)

iii) Device drivers


It is a special kind of software that controls a specific hardware device attached to the system. Device drivers are essential for a computer to work properly. A device driver usually communicates with the hardware by means of the communications subsystem or computer bus to which the hardware is connected. A device driver acts as a translator between the hardware device and the programs or operating systems that use it. Hardware devices that need a driver to connect to a system include displays, sound cards, printers, mice and hard disks.

Examples of device drivers;

  • BIOS Driver
  • Display Drivers
  • Motherboard Drivers
  • Printer Drivers
  • ROM Drivers
  • Sound card Driver
  • USB Drivers
  • USB Drivers
  • VGA Drivers
  • VGA Drivers
  • Virtual Device Drivers

iv) Firmware

Firmware is a permanent software which is embedded in the system’s read-only memory. It is a set of instructions which are permanently stored onto the hardware device. It offers vital information regarding how a particular device interacts with different other hardware. Some examples of firmware include:

  • Embedded Systems
  • BIOS
  • UEFI
  • Computer Peripherals

2. Application Software



Application software is a set of programs designed to perform a specific task. Application software may consist of a single program, such as Microsoft's notepad for writing and editing a simple text. It may also consist of a collection of programs, often called a software package, which work together to accomplish a task, such as a spreadsheet package. Applications on a computer cannot run on their own; they require a computer's OS, along with other supporting system software programs, to work.

They are also known as end-user programs or even productivity programs designed for a specific function. These software are often developed through custom software development, based on the requirements of the users. Examples of application software include:

i) Word processors

This software allows users to create, edit, format, and manipulate the text and more. It offers lots of options for writing documents, creating images, and more. Examples include:

  • MS Word
  • Apple iWork-Pages
  • Google docs
  • Corel WordPerfect

ii) Web browsers

These applications are used to browse the internet. They help the user in locating and retrieving data across the web. Examples include:

  • Google chrome
  • Internet Explorer
  • Mozilla Firefox
  • UC Browser
  • Safari

iii) Database software

his software is used to create and manage a database. They organize data in the database. It is also called a Database Management System(DBMS). Examples if DBMS incllude:

  • MySQL
  • FileMaker
  • MS Access
  • FoxPro
  • Clipper

iv) Multimedia software

It is a software that is use to play, create or record images, audio or video files. These software are used for animation, video editing, graphics as well as image editing. Due to the high demand for such software, every software product development company has vast avenues in developing them. Examples include:

  • Adobe Photoshop
  • Windows Movie Maker
  • Picasa
  • VLC Media Player

Other types of software include:

A) Programming Software


It is a collection of tools that help developers in writing other software or programs. It assists them in creating, debugging, and maintaining software or programs or applications. It translate programming language such as Java, C++, Python, etc., into machine language code. So, it is not used by end-users. For example, compilers, linkers, debuggers, interpreters, text editors. Examples of programming software include:

  • Sublime text: It is a cross-platform code editor for Linux, Mac, and Windows.
  • Coda: It is a programming language editor for Mac.
  • Eclipse: It is a java language editor.
  • Notepad++: It is an open-source editor for windows.

B) Middleware



The term middleware describes software that mediates between application and system software or between two different kinds of application software. For example, middleware enables Microsoft Windows to talk to Excel and Word. Data management, application services, messaging, authentication, and API management are all commonly handled by middleware. Middleware helps developers build applications more efficiently. It acts like the connective tissue between applications, data, and users. Examples of middleware include:

  • Game engine: Game engine middleware is more helpful for game developers with accessing to different kinds of tools which help to make game process easier. For examples are game scripting, physics simulations, and graphics rendering.
  • Integration middleware:This middleware helps to deliver better integration framework, and with the help of it various operations, executions and runtime services can be determined and handled from various apps.
  • Device middleware:This middleware helps to deliver group of tools which are implemented to develop various applications, and they can be run in particular hardware environment.
