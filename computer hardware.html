What is Computer Hardware ?

Computer hardware is a collective term used to describe any of the physical components of an analog or digital computer. This distinguishes it from software, which consists of written instructions that tell the physical components what to do. Without any hardware, your computer would not exist, and software could not be used. Computer hardware may sometimes be seen abbreviated as computer HW.

The components that make up hardware can be categorized as being either internal or external. Internal components are those installed inside the computer. External components are connected to the outside of the computer, they can also be referred to as peripherals, or peripheral devices.

Internal hardware components

Below is a list of hardware components found inside a computer.

1. Motherboard


A motherboard is an electronic circuit board in a computer which interconnects hardware devices attached to it. It can be thought of as the "mother" that holds all the pieces together. It allocates power to all hardware located on it and enables them to communicate with each other. It is meant to hold the computer's microprocessor chip and let other components connect to it. Each component that runs the computer or improves its performance is part of the motherboard or connected to it through a slot or port.

Components of a motherboard include;

  • CPU slot
  • RAM slot
  • Expansion slot
  • Capacitor
  • Northbridge
  • Inductor
  • PCI slot
  • USB port
  • AGP slot
  • Heat sink
  • CMOS battery
  • Power connector

2. Central Processing Unit (CPU)

It is the electronic circuitry within a computer that exercutes instructions that make up a computer program. It performs basic arithmetic, logic, controlling, and input/output operations specified by the instructions in the program. It performs most of the calculations which enable a computer to function, and is sometimes referred to as the “brain” of the computer. Each CPU has a clock speed, which is the number of instructions it can process in any given second, measured in gigahertz.

Intel and AMD are the two most popular CPU manufacturers for desktops, laptops, and servers, while Apple, NVIDIA, and Qualcomm are big smartphone and tablet CPU makers.

3. Random Acess Memory (RAM)

Random Access Memory (RAM) is the physical hardware inside a computer that temporarily stores data, serving as the computer's "working" memory. As the main memory of a computer, RAM is much faster to read from and write to than other types of storage, including a hard disk drive (HDD), solid-state drive (SSD) and optical drive. RAM is volatile, which means that data stays in RAM if the computer is on, but it's lost when the computer is turned off.

It is normally found in the memory slots of the motherboard. Generally speaking, the faster the RAM, the faster the processing speed at which memory moves data to other components. Some popular manufacturers of RAM include Kingston, PNY, Crucial, and CORSAIR.

4. Power Supply

The power supply unit is the piece of hardware that converts the power provided from the outlet into usable power for the many parts inside the computer case. It converts alternating current to direct current that the computer components require. It also regulates the voltage to an adequate amount, which allows the computer to run smoothly without overheating. Typically, more power is needed to run more complex systems.

5. Video card

Video card is an expansion card that enables the computer to send output images to a video display device such as a monitor,TV, or projector. The video card typically installs through a slot on the motherboard. It is also known as a display adapter, graphics card, video adapter, video board, or video controller.

Video cards are used by gamers in place of integrated graphics due to their extra processing power and video ram. A staggering number of companies manufacture video cards, but almost every one includes a graphics processing unit (GPU) from either NVIDIA Corporation or AMD.

6. Hard Drive(HDD)

The hard disk drive is the main data storage device of a computer. The operating system, software titles, and most other files are stored in the hard disk drive.The hard drive is sometimes referred to as the "C drive" because Microsoft Windows, by default, designates the "C" drive letter to the primary partition on the primary hard drive in a computer. Unlike RAM, hard drives are non-volatile, which means that they maintain their stored data even when powered off.

7. Solid State Drive(SSD)

A type of nonvolatile storage device that stores persistent data on solid-state flash memory. Unlike a hard drive, an SSD has no moving parts, which gives it advantages, such as faster access time, noiseless operation, higher reliability, and lower power consumption.

8. Optical Drive

An optical disc drive (ODD) uses laser light or electromagnetic waves to read or write data onto optical discs. This type of drive allows a user to retrieve, edit and delete the content from optical disks such as CDs, DVDs and Blu-ray disks. The optical drive normally goes by other names like a disc drive, CD drive, DVD drive, or BD drive.

9. Card Reader

Also known as a media card reader is a hardware device for reading and writing data on a memory card such as a multimedia card. Some card readers only have one card slot, and some have multiple card slots for different cards and media. Most card readers accept multiple memory card formats, including compact flash (CF), secure digital (SD), and Sony's Memory Stick. Some card readers accept various other formats such as XD, SmartMedia, Microdrive, and Memory Stick Pro Duo cards.

It mostly find use for those who want to download information from expanded storage on a smartphone, or memory from a digital camera.

External Hardware Components

Below is a list of components found outside a computer, although many tablets, laptops, and netbooks integrate some of these items into their housings:

1. Monitor

A monitor is a hardware device that displays the video and graphics information generated by the computer via the video card. Monitors are similar to TVs but it displays information at a much higher resolution. A monitor is sometimes referred to as a screen, display, video display, video display terminal, video display unit, or video screen.

Types of monitors

  • CRT Monitor: It has cathode ray tubes which produce images in the form of video signals. Its main components are electron gun assembly, glass envelope, fluorescent screen, deflection plate assembly, and base.

  • LCD Monitor:It is a flat panel screen. It uses liquid crystal display technology to produce images on the screen. Advanced LEDs have thin-film transistors with capacitors and use active-matrix technology, which allows pixels to retain their charge.

  • LED Monitor: It is an advanced version of an LCD monitor. Unlike an LCD monitor, which uses cold cathode fluorescent light to backlight the display, it has LED panels, each of which has lots of LEDs to display the backlight.

  • Plasma Monitor: It uses plasma display technology that allows it to produce high resolutions of up to 1920 X 1080, wide viewing angle, a high refresh rate, outstanding contrast ration, and more.

2. Keyboard

A keyboard is a piece of computer hardware used to input text, characters, and other commands into a computer or similar device. A keyboard is one of the primary input devices used with a computer. Similar to an electric typewriter, a keyboard is composed of buttons used to create letters, numbers, and symbols, and perform additional functions. Today, most desktop computer keyboards connect to the computer using either USB or Bluetooth for wireless communication. Types of keyboards include; QWERTY Keyboards, AZERTY Keyboards, and DVORAK Keyboards.

Microsoft and Logitech are probably the most popular physical keyboard manufacturers, but many other hardware makers also produce them.

3. Mouse

The computer mouse is a hand-held pointing device used to manipulate objects on a computer screen. Whether the mouse uses a laser or ball, or is wired or wireless, a movement detected from the mouse sends instructions to the computer to move the cursor on the screen in order to interact with files, windows, and other software elements. Types of Mouse include; Cordless (Wireless), Footmouse, IntelliMouse (Wheel mouse), J-Mouse, Joystick, Mechanical, Optical, Touchpad (Glidepoint), Trackball, and TrackPoint.

Basic functions of a mouse

  • Open or execute a program: It allows you to open a folder or document and execute a program. You are required to take the cursor on the folder and double click it to open it.

  • Hovering: Hovering is an act of moving the mouse cursor over a clickable object. During hovering over an object, it displays information about the object without pressing any button of the mouse.

  • Move the cursor: It is the main function of the mouse; to move the cursor on the screen.

  • Scroll: It allows you to scroll up or down while viewing a long webpage or document.

  • Select: It allows you to select text, file, or any other object.

4. Printer

A printer is an external hardware output device that takes the electronic data stored on a computer or other device and generates a hard copy. Printers are one of the most popular computer peripherals and are commonly used to print text and photos. The two most common types of printers are inkjet and laser printers. Inkjet printers are commonly used by consumers, while laser printers are a typical choice for businesses. Printers also have other functions besides printing, this includes copying, faxing and scanning making it a useful piece of hardware.

5. Speakers

A computer speaker is an output hardware device that connects to a computer to generate sound. The signal used to produce the sound from the computer speaker is created by the computer's sound card. It is used for listening to music, movies, and other audio.

Speakers are transducers that convert electromagnetic waves into sound waves. The speakers receive audio input from a device such as a computer or an audio receiver. This input may be either in analog or digital form. Analog speakers simply amplify the analog electromagnetic waves into sound waves. Since sound waves are produced in analog form, digital speakers must first convert the digital input to an analog signal, then generate the sound waves.

6. Projector

A projector is an output device that takes images generated by a computer or Blu-ray player and reproduce them by projection onto a screen, wall, or another surface. In most cases, the surface projected onto is large, flat, and slightly colored. Modern digital projectors are often used for watching movies, to enhance presentations, or as a teaching aid. They connect to a computer via the HDMI port, or VGA ports for older computers.

7. External hard drive

An external drive is just a hard drive (HDD) or solid state drive (SSD) that is connected to a computer on the outside rather than on the inside. Some draw power from the computer via the data cable, others need an AC wall connection. The main benefit of an external drive is their portability, you can carry relatively large amounts of data around with you, or transfer data between different computers. External hard drives come in varying storage capacities, but they all connect to a computer either by USB, FireWire, eSATA, or wirelessly.

8. Scanner

A scanner is an input device that transfers images or text to a computer using optical technology. The computer converts the signal to a digital image, enabling it to be edited, emailed, or printed. For example, a scanner may be used to convert a printed picture, drawing, or document (hard copy) into a digital file which can be edited on a computer.

9. USB flash drive

A USB flash drive is a portable data storage device. Unlike a standard hard drive, the flash drive has no movable parts; it contains only an integrated circuit memory chip that is used to store data. It is also referred to as a USB flash drive, data stick, pen drive, memory unit, keychain drive, and thumb drive. It is often the size of a human thumb (hence the name), and connects to a computer via a USB port. Flash drives are an easy way to store and transfer information between computers.

10. Joystick

A joystick is an input device that control a character or machine in a computer program. Joysticks are also used to control various real life vehicles, such as airplanes, trucks, wheelchairs, surveillance cameras, and unmanned underwater vehicles, which are increasingly computerized. Today, most computer joysticks connect to the computer using a USB port.
